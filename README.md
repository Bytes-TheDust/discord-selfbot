# Setup

## 1. Requirements
 - node.js
 - discord.js-v13-self
 - axios
 - fs

## 2. Clone The Repo

## 3. Installation
 - Install Node.js For Your Server
 - Run The Following Command
```
npm install discord.js-selfbot-v13@latest axios fs
```

## 4. Run index.js in discord-selfbot/src/index.js
```
cd discord-selfbot
```
```
cd src
```
```
node index.js
```
