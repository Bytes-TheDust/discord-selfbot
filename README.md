# Requirements
 - node.js
 - discord.js-v13-self
 - axios
 - fs


# Setup

## 1. Clone The Repo
```
git clone https://github.com/Bytes-TheDust/discord-selfbot.git
```

## 2. Token & Prefix
 - Edit The Value In config/config.json
```
{
  "token": "YOUR_DISCORD_TOKEN",
  "prefix": "$"
}
```
## 2. Installation
 - Install Node.js & npm For Your Server
 - Run The Following Command
```
npm install discord.js-selfbot-v13@latest axios fs
```

## 3. Run index.js in src/index.js
```
cd discord-selfbot
```
```
cd src
```
```
node index.js
```
